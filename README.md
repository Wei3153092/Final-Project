# 📘 Data Structures & Algorithms Study Notes
> 元智大學 (YZU) 資料結構課程學習筆記與重點整理

這個儲存庫（Repository）收錄了資料結構課程的核心教材整理與個人筆記，涵蓋從基礎演算法概念到複雜的圖形理論與雜湊技術。

---

## 📂 章節導覽 (Table of Contents)

### 1. 基礎與線性結構 (Basics & Linear Structures)
* **05 Algorithms**: 演算法定義、五大準則（輸入、輸出、明確性、有限性、有效性）。
* **08 Arrays**: 連續記憶體配置、隨機存取 ($O(1)$) 與插入刪除的成本。
* **09 Linked Lists**: 單向與雙向鏈結串列、動態記憶體分配。
* **10 Stacks & Queues**: LIFO 與 FIFO 原則、環狀隊列實作。

### 2. 進階非線性結構 (Advanced Non-linear Structures)
* **11 Hashing**: 雜湊函數設計、衝突處理（Chaining, Open Addressing）。
* **12 Trees**: 二元搜尋樹 (BST) 與 AVL 樹的自平衡機制。
* **13 Heaps**: 最大堆積與最小堆積、優先隊列應用。
* **14 Graphs**: 圖形走訪 (DFS/BFS)、相鄰矩陣與相鄰串列。
